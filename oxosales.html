Para construir el nombre de estas sustancias se cita primero el oxoanión, se añade luego la preposición "de" y se agrega el nombre del catión. Para indicar las cantidades que hay de cada parte se utilizan prefijos multiplicadores.

Los prefijos multiplicadores para el oxoanión son bis (2), tris (3), tetrakis (4), pentakis (5) y así sucesivamente. Para el catión se siguen usando los prefijos habituales.

Como el nombre del oxoanión puede estar en cualquiera de las maneras posibles, una misma fórmula puede recibir nombres diferentes. Todos son igualmente válidos.

Fíjate en los siguientes casos:

Al2(CrO4)3 tris(tetraoxidocromato) de dialuminio, tris(cromato) de dialuminio

NiPO4 fosfato de níquel, tetraoxidofosfato de níquel

Ca(HCO3)2 bis[hidrogeno(trioxidocarbonato)] de calcio, bis(hidrogenocarbonato) de calcio
