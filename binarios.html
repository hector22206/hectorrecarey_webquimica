Para nombrar las sales binarias, se escribe primero el elemento no metálico añadiendo la terminación -uro, posteriormente se plasma el elemento metálico. Por ejemplo, el sodio (Na) se combina con el flúor (F) para formar fluoruro de sodio (NaF). Para nombrar la sal binaria, se coloca el número de valencia en notación romana entre paréntesis, después del nombre de la especie. A continuación, se presentan algunos ejemplos de sales binarias:

NaCl: Cloruro de sodio

CaBr2: Dibromuro cálcico

FeCl3: Tricloruro de hierro
